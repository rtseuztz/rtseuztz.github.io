## Smart Resume Generation at Your Fingertips


![Rezumake Dashboard](/assets/images/resume_dashboard.png)

An intelligent tool that helps you create tailored resumes for different job applications in minutes. Rezumake innovates the job application process by streamlining resume creation to save you time and increase your chances of landing interviews.

Built with modern technologies including **Angular**, **Java SpringBoot**, and **Google Cloud Platform**, this full-stack application demonstrates enterprise-level architecture and best practices.

## ‚ú® Key Features

### 1. Secure Authentication System

![User Signup](/assets/videos/CreateAccount.gif)

Rezumake features a **custom-built authentication system** with enterprise-grade security:

- **Encrypted Data Storage**: All user personal information is encrypted in the database
- **Phone Verification**: Integrated with Twilio for SMS verification
- **JWT Token Authentication**: Secure frontend authentication using JSON Web Tokens
- **Free Trial System**: Full access to paid features during trial period

**Technologies Used**: JWT, Twilio API, BCrypt encryption

---

### 2. Comprehensive Profile Management

![Personal Details](/assets/videos/PersonalDetails.gif)

Store and manage your professional information with an intuitive, user-friendly interface:

- **Multiple Sections**: Education, work experience, projects, skills, and more
- **Rich Text Editing**: Format your content with ease
- **Reusable Components**: Save time by reusing information across multiple resumes
- **Real-time Validation**: Instant feedback on required fields

The sleek UI makes it easy to organize your professional history, which serves as the foundation for generating tailored resumes.

---

### 3. AI-Powered Resume Generation

![Resume Generation](/assets/videos/GenerateResume.gif)

Create professional resumes in seconds with intelligent customization:

- **Generic Resumes**: Generate a comprehensive resume with all your information
- **Job-Tailored Resumes**: Customize content based on specific job descriptions
- **Multiple Templates**: Choose from various professional designs
- **PDF Export**: Download your resume in industry-standard format
- **Version History**: Track and manage different resume versions

Simply input your target job details, and the system intelligently selects and formats the most relevant information from your profile.
